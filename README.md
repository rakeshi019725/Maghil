# Offline Task / Order Management App (React Native)

## Objective
This application demonstrates an offline-first approach using React Native.
Users can create tasks/orders without internet connectivity and the app
automatically syncs pending records once the network is restored.

## Tech Stack
 React Native (TypeScript)
 Redux Toolkit (State Management)
 AsyncStorage (Local Persistence)
 NetInfo (Network Detection)
 React Navigation

## Features
 Create tasks while offline
 View task list with sync status
 Auto sync when internet is available
 Manual retry for failed sync
 Persistent local storage
 Clean and scalable folder structure
